# Source Filings

Place FERC Form 1 source files here (`.html` iXBRL renders or `.xbrl` / `.xml` instances).

Example layout:

```
filings/
  ClecoPowerLlc2023.html
  ClecoPowerLlc2020.xbrl
  SouthwesternElectricPowerCompany2022.html
```

Source filings are **not** committed by default (large and publicly available from FERC eCollection).
